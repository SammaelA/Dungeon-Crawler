Инструкция по сборке на Windows с помощью Visual Studio
1) Настройки параметров проекта соответствуют Windows 10 и Visual Studio 2017
при использовании другой версии системы или среды требуется изменить настройки проекта
2) В параметрах проекта (каталоги VC++) при необходимости указать путь до библиотеки
sfml (папка include из данного архива) и SDK для Windows (нужно для заголовочного файла
Windows.h)
3)В диспетчере конфигураций выбрать конфигурацию Release
4)Запустить сборку проекта
